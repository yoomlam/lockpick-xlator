# Output Rulesets

Generated rulesets in target rules engine format.

## Purpose
This folder contains the transpiled/converted outputs from the specs:
- Rules engine-specific formats (e.g., Drools DRL, OpenFisca Python)
- Decision tables
- Rule configuration files

## Generation
These files are automatically generated from specs in `specs/ruleset/` directory.
Do not manually edit these files - update the source specs instead.
